Association Measures

Given a concept, the goal is to extract concept links most associated with the source concept. We do this by analyzing the statistical distribution properties across all concepts, and ranking the concepts through an association measure. There are many association measures one can use, all having different properties with different behaviors in different situations. Here, we use different association measures based on distributional frequencies. There are other measures one can use based on comparing the context of words surrounding the occurrence of two concepts. However, we have not included these at this time due to their computational cost.

The moving text window for measuring association can be set to any length, and here we use sentences as boundaries for this window. The table below represents a contingency table representing observed co-occurrence frequencies within a certain text window for concepts x and y.

x represents presence of the concept and \lnot{x} represents the absence of the concept.

N f(y) f(\lnot{y})
f(x) f(x, y)=a f(x, \lnot{y})=b
f(\lnot{x}) f(\lnot{x}, y)=c f(\lnot{x}, \lnot{y})=d

  • Conditional probability    P(y|x)=\frac{a}{a+b}
  • Mutual dependence    \log{\frac{P(x,y)^2}{P(x)P(y)}}=\log{\frac{a^2}{(a+b)(a+c)}}
  • Normalized expectation    \frac{2P(x,y)}{P(x)+P(y)}=\frac{2a}{2a+b+c}
  • Mutual expectation    \frac{2P(x,y)}{P(x)P(y)}P(x,y)=\frac{2a}{2a+b+c}\left(\frac{a}{N}\right)
  • Pearson phi coefficient    \frac{ad-bc}{\sqrt{(a+b)(a+c)(b+d)(c+d)}}
  • Student t test    \frac{f(x,y)-\hat{f}(x,y)}{\sqrt{f(x,y)\left[1-\frac{f(x,y)}{N} \right]}}=\frac{a-\frac{(a+b)(a+c)}{N}}{\sqrt{a-\frac{a^{2}}{N}}}
  • Z score    \frac{f(x,y)-\hat{f}(x,y)}{\sqrt{\hat{f}(x,y)\left[1-\frac{\hat{f}(x,y)}{N} \right]}}=\frac{a-\frac{(a+b)(a+c)}{N}}{\sqrt{\frac{(a+b)(a+c)}{N}\left[1-\frac{(a+b)(a+c)}{N^{2}} \right]}}
  • Russel-Rao    \frac{a}{a+b+c+d}
  • Jaccard    \frac{a}{a+b+c}
  • First Kulczynsky    \frac{a}{b+c}
  • Second Sokal-Sneath    \frac{a}{a+2(b+c)}
  • Driver-Kroeber    \frac{a}{\sqrt{(a+b)(a+c)}}
  • Fager    \frac{a}{\sqrt{(a+b)(a+c)}}-\frac{1}{2}\max(b,c)
  • Baroni-Urbani    \frac{a+\sqrt{ad}}{a+b+c+\sqrt{ad}}
  • Michael    \frac{4(ad-bc)}{(a+d)^{2}+(b+c)^{2}}
  • Piatersky-Shapiro    P(x,y)-P(x)P(y)=\frac{a}{N}-\frac{(a+b)(a+c)}{N^2}
  • Poisson-Stirling    a\left(\log{\frac{aN}{(a+b)(a+c)}}-1\right)
